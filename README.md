
# Advanced Portfolio with CSS Grid

## Project Overview
This portfolio is built using advanced CSS techniques including CSS Grid, Flexbox, animations, and CSS variables.

## Setup
1. Download or clone repository
2. Open index.html in browser

## Features
- CSS Grid layout
- Flexbox navigation
- Theme switching with CSS variables
- Smooth animations
- BEM methodology
